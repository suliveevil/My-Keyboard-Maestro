*** 修饰符号 prefix/suffix

类似修饰键的作用。
推荐使用 ~重复的字符~ 或者 <kbd>Space</kbd> 或者 <kbd>-</kbd> 或者 <kbd>=</kbd>。
中英文输入法下, 只有数学符号 <kbd>-</kbd> 和<kbd>=</kbd> 是一样的(数字可能需要参与选择文本,不予考虑)。注意只可用作前缀，因为这两个键在中文输入法中是翻页键。
